Data Types & Entry Procedures

Specification of data formats, types, and entry requirements for each indicator category.

HAZARD Dimension

Natural and human-induced hazards that may cause loss of life or damage.

Natural Hazards - Numeric

Data Type: numeric

Format: Decimal numbers (e.g., 5.2, 10.0)

Examples:

  • Drought Frequency: events per decade (0-20)
  • Earthquake Exposure: MMI scale (0-10)
  • Storm Surge: meters (0-10)

Exposure Indicators - Percentage

Data Type: percentage

Format: 0-100 (no % symbol)

Examples:

  • Flood Exposure: % of population exposed
  • Sea Level Rise: % of coastal population
  • Heat Wave Exposure: % affected annually

Count Indicators

Data Type: count

Format: Whole numbers

Transformation: LOG (for large ranges)

Examples:

  • Violence Events: number of incidents
  • Fatalities: number of deaths
  • Cyclone Tracks: number of tracks

VULNERABILITY Dimension

Socio-economic and demographic factors that increase susceptibility to hazards.

Index Indicators

Data Type: index

Format: 0-1 scale (normalized indices)

Examples:

  • HDI: Human Development Index (0-1)
  • MPI: Multidimensional Poverty Index (0-1)
  • GINI: Income inequality coefficient (0-1)

Note: Some indices are inverted (higher = less vulnerable)

Rate Indicators

Data Type: numeric

Format: Per 1,000 or per 100,000 population

Examples:

  • Infant Mortality: per 1,000 live births
  • TB Incidence: per 100,000 population
  • Malaria Prevalence: per 1,000 population

IPC Phase Classification

Data Type: categorical

Format: Integer 1-5

Values:

  • 1 = Minimal
  • 2 = Stressed
  • 3 = Crisis
  • 4 = Emergency
  • 5 = Famine

COPING CAPACITY Dimension

Infrastructure and institutional capacity to cope with and recover from disasters.

Coverage Indicators

Data Type: percentage

Format: 0-100

Examples:

  • Immunization Coverage: % of children vaccinated
  • Electricity Access: % with connection
  • Water Access: % with safe water

Note: Higher values = better coping capacity

Density Indicators

Data Type: numeric

Format: Per population ratio

Examples:

  • Physicians: per 10,000 population
  • Health Facilities: per 10,000 population
  • Road Density: km per sq km

Governance Indicators

Data Type: index

Format: Varies by source

Examples:

  • Government Effectiveness: -2.5 to +2.5 (WGI)
  • Sendai Framework Progress: 0-1
  • Social Cohesion: 0-1

Data Entry Procedures

  1. Select Resolution Level: National, ADM1 (Regional), or ADM2 (District)
  2. Choose Indicator: Select from the appropriate dimension and category
  3. Enter Raw Value: Use the specified unit and format for the indicator
  4. Specify Time Period: Year and optionally quarter
  5. Provide Source: Document the data source (e.g., "DHS 2022", "Field Survey")
  6. Submit for Review: Data is automatically normalized and sent for verification
